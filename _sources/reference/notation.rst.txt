| Code | Type          | Purpose                      | Example                      | Atom | Non-atom |

| c    | concept       | Define atomic concepts       | apple/c                      | x    | x        |
+------+---------------+------------------------------+------------------------------+------+----------+
| p    | predicate     | Build relations              | (*is/p* berlin/c nice/c)     | x    | x        |
+------+---------------+------------------------------+------------------------------+------+----------+
| a    | auxiliary     | Modify a predicate           | (*not/a* is/p)               | x    |          |
+------+---------------+------------------------------+------------------------------+------+----------+
| m    | modifier      | Modify a concept             | (*red/m* shoes/c)            | x    | x        |
+------+---------------+------------------------------+------------------------------+------+----------+
| b    | builder       | Build concepts from concepts | (*of/b* capital/c germany/c) | x    |          |
+------+---------------+------------------------------+------------------------------+------+----------+
| w    | meta-modifier | Modify a modifier            | (*very/w* large/m)           | x    |          |
+------+---------------+------------------------------+------------------------------+------+----------+
| x    | subpredicate  | Auxiliary predicate          | (*by/x* john/c)              | x    |          |
+------+---------------+------------------------------+------------------------------+------+----------+
| t    | trigger       | Build specifications         | (*in/t* 1994/c)              | x    |          |
+------+---------------+------------------------------+------------------------------+------+----------+
| r    | relation      | Express facts, statements,   | *(is/p berlin/c nice/c)*     |      | x        |

|      |               | questions, orders, ...       |                              |      |          |

| s    | specifier     | Relation specification       | *(in/t 1976/c)*              |      | x        |
